About

The Rareware Collectathons refer to the three platformer collectathons made by Rareware: Banjo-Kazooie, Banjo-Tooie, and Donkey Kong 64.

There are three main race catagories that are run with these games: Any%, Restricted Any%, 301%

Currently there is only one planned race which is the 301% race. This is Banjo-Kazooie 100%, Banjo-Tooie 100%, and Donkey Kong 64 101% back to back to back!
